🍽️ Recipe Finder

A modern and responsive Recipe Finder web application built using HTML, CSS, and JavaScript. The application uses TheMealDB API to search and display recipes dynamically with an interactive and user-friendly interface.

🚀 Features
🔍 Search recipes by name
🍽 Browse recipes by category
🎲 Random recipe generator
❤️ Save favorite recipes (Local Storage)
📖 View recipe details in a modal
🥗 Ingredients list with cooking instructions
▶️ YouTube recipe tutorial link
🌙 Dark Mode
⏳ Loading Spinner
🔔 Toast Notifications
⌨️ Search using Enter key
⬆️ Back to Top button
📱 Fully Responsive Design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Font Awesome
TheMealDB API
Local Storage
📖 API Used
